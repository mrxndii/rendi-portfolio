# Muhammad Renaldi - Personal Portfolio

This repository contains the source code for my personal portfolio website. It is designed to showcase my projects, technical skills, and background. 

The website is a static single-page application built entirely from scratch without heavy frameworks to ensure maximum performance and customizability.

## Features

- Custom interactive UI with CSS Grid and Flexbox layouts.
- Built-in audio player for background music.
- Multilingual support (Indonesian and English) managed via JavaScript logic.
- HTML5 Canvas implementation for background animations.
- Responsive design tailored for mobile, tablet, and desktop environments.
- Security implementations including Content Security Policy (CSP) meta tags, anti-scraping scripts, and form honeypots.

## Technologies Used

- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- [Lucide Icons](https://lucide.dev/) for UI elements
- [Devicon](https://devicon.dev/) for technology stack logos

## Running Locally

Since this is a static website, you can run it without any complex build steps or Node.js dependencies.

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/your-repository.git
   ```
2. Open the project folder.
3. Serve the directory using a local web server. For example, if you have Python installed:
   ```bash
   python -m http.server 8000
   ```
   Or simply use the **Live Server** extension in VS Code.
4. Navigate to `http://localhost:8000` in your browser.

## Project Structure

- `/assets` - Static assets including images, icons, and audio files.
- `/css` - Styling definitions, including CSS variables for theming.
- `/js` - Core logic for DOM manipulation, canvas rendering, and translation mapping.
- `index.html` - The main entry point of the application.
